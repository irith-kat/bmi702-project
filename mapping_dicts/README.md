# mapping_dicts

Vocabulary and code mapping files used by the preprocessing pipeline.

| File | Source | Used by |
|---|---|---|
| `Phecode_map_v1_2_icd9_icd10cm.csv` | [PheWAS Catalog](https://phewascatalog.org/phewas/#phe12) | `rollup_icd_to_phecode` |
| `CCS_Services_Procedures_v2025-1_052425.csv` | [AHRQ CCS](https://hcup-us.ahrq.gov/toolssoftware/ccs_svcsproc/ccscpt_downloading.jsp) | `rollup_cpt_to_ccs` |
| `gcpt_drug_ndc.csv` | [MIMIC-IV OMOP CDM](https://github.com/OHDSI/MIMIC) | `rollup_ndc_to_ingredient` (fallback) |
| `d_labitems_to_loinc.csv` | [MIT-LCP mimic-code](https://github.com/MIT-LCP/mimic-code/tree/main/mimic-iv/mapping) | `rollup_itemid_to_loinc` |
| `inputevents_to_rxnorm.csv` | [MIMIC-IV OMOP CDM](https://github.com/OHDSI/MIMIC) | future ICU drug rollup |
| `ndc_to_rxnorm_ingredient.csv` | generated by `build_ndc_to_rxnorm.py` | `rollup_ndc_to_ingredient` |
| `drug_name_to_rxnorm_ingredient.csv` | generated by `build_ndc_to_rxnorm.py` | `rollup_ndc_to_ingredient` (fallback) |
| `athena_ndc_rxnorm_export/` | [OMOP Athena](https://athena.ohdsi.org/) — NDC + RxNorm vocabularies | input to `build_ndc_to_rxnorm.py` |

## Regenerating the NDC mapping

If the Athena export is updated, re-run:

```bash
uv run python mapping_dicts/build_ndc_to_rxnorm.py
```

See `athena_ndc_rxnorm_export/README.md` for download instructions.

NOTE: the mapping dicts are stored in github inside the `mapping_dicts_csvs.zip` file to reduce clutter. Unzip them before running the pipeline.
